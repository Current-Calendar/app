# Contrato de las APIs

## Propósito

El presente documento tiene la intención de informar a los desarrolladores de Current sobre la arquitectura de API REST del proyecto. A continuación se encuentran las rutas a emplear separadas por módulo. 

## Gestión de usuarios

| Ruta | Descripción |
| :---: | :---: |
|/auth/register/|Registro de nuevo usuario|
|/auth/login/|Inicio de sesión|
|/auth/logout/|Cierre de sesión|
|/auth/password_reset/|Solicitar email de restablecimiento|
|/auth/password_reset/confirm/|Cambiar contraseña con el token del email|
|/users/me/|Ver/Editar/Borrar perfil propio|
|/users/{id}/|Ver perfil de un usuario|
|/users/{id}/follow/|Seguir/Dejar de Seguir a un usuario|
|/users/{id}/block/|Bloquear a un usuario|


## Gestión de calendarios personales

| Ruta | Descripción |
| :---: | :---: |
|/calendars/|Listar mis calendarios y crear uno nuevo|
|/calendars/{id}/|Detalle, edición o borrado de un calendario|
|/calendars/all_events/|Vista unificada de todos los eventos del usuario|
|/events/|Crear evento|
|/events/{id}/|Detalle, edición o borrado de evento|
|/calendars/import/|Importar calendario|
|/calendars/{id}/export/|Exportar calendario|

## Gestión de calendarios publicos

| Ruta | Descripción |
| :---: | :---: |
|/calendars/{id}/comments/ |Ver y publicar comentarios|
|/event/{id}/save/|Guardar evento público a mi calendario personal|
|/public/explore/|Feed de eventos recomendados|
|/public/search/|Búsqueda por palabra clave, etiqueta o flitro|
|/calendars/{id}/report/|Reportar calendario|
|/calendars/{id}/stats/|Estadísticas del calendario|
|/calendars/{id}/mute/|Silenciar calendario|
|/events/{id}/report/|Reportar evento|

## Gestión de chats

| Ruta | Descripción |
| :---: | :---: |
|/chat/conversations/|Listado de chats|
|/chat/{user_id}/|Ver historial de mensajes o mandar un mensaje|

## Gestión de notificaciones

| Ruta | Descripción |
| :---: | :---: |
|/notifications/|Centro de notificaciones|
|/users/me/settings/|Configurar notificaciones|

## Administración

| Ruta | Descripción |
| :---: | :---: |
|/admin/users/{id}/|CRUD de usuarios para administrador|
|/admin/reports/|Ver y gestionar reportes de contenido ofensivo|
|/admin/users/{id}/suspend/|Suspender cuenta de usuario|
|/admin/metrics/|Ver métricas del sistema|
