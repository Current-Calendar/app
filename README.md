## Getting started  

Designed for management and social interaction, Current is a calendar-centered application in which you can create and manage your own calendars, share them with colleages for them to contribute, or just publish them so
others can see what you're up to.
- Visit our [Landing page](https://currentcalendar.vercel.app/) to join the Beta
